# Retail-Inventory-Manager

A Flask web application for managing retail inventory, designed with a BCNF-normalized database to ensure data integrity and reduce redundancy. This project allows users to create, read, update, and delete inventory items, as well as search for optimum stock efficiently.

## Features

- **Inventory Management:** Add, update, and remove items from the database.  
- **Search Functionality:** Find optimum stock or specific items.  
- **Database Integrity:** Uses a BCNF-normalized schema with foreign key relationships.  
- **User-Friendly Interface:** Built with Flask, HTML templates, and styled with CSS/Bootstrap.
